Markdown语法总结
2020-03-18|Technology
Word count: 594|Reading time: 2 min

推荐编辑器：Typora，各类操作系统官网都有相应的版本，很干净，超好用。

Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。Markdown具有一系列衍生版本，用于扩展Markdown的功能（如表格、脚注、内嵌HTML等等）。

一、语法

快捷键 Markdown 效果
Ctrl+B *text* text，加粗
Ctrl+L *text* text，斜体
Ctrl+Alt+U ~~text~~ text，删除线
Ctrl+K [title](http://) title，链接
Ctrl+Shift+K `code|code`，行内代码
Ctrl+Shift+L ![alt](http://) 显示图片
Ctrl+L * item * item，列表（下方有展示）
Ctrl+Q > quote > quote，引用
# heading # heading，一级标题
Ctrl+H ## heading ## heading，二级标题
Ctrl+HH ### heading ### heading，三级标题

参考：Markdown_百度百科

  • 标题：共有六级，按照上方格式使用
  • 列表：可以有多级标题，左侧小圆点即为一级标题，下方是其他各级标题：
    • 二级标题
      • 三级标题
        • 四级标题
  • 引用：Hexo博客显示不出来，因为不常用，所以没去解决。

二、插入图片或视频

1、图片

  • 方法一：上方表格中的方法
  • 方法二：<img src = “http://” width = “50%” height = “50%”/>，就是内嵌的HTML，可以更改图片大小

2、视频

  • 方法一：<iframe weight=“500” height=“300” src=“http://” autoplay=“false”></iframe>
  • 方法二：<video src=“http://” width=“100%” height=“400” controls=“controls” frameborder=“0”></video>

四、代码

1、行内代码：上方表格中的方法

2、行间代码块

另起一行英文状态下输入 ``` (制表符上方的符号)，然后回车，再弹出的代码文本框右下角选择代码语言。

或者直接输入 ```python ，然后回车，弹出python代码的文本框进行输入。
